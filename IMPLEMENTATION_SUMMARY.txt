2. HAINES INDEX
   File: src/haines_index.H
   Lines: 380
   Purpose: Atmospheric stability for plume-dominated fire potential
   
   Key Functions:
   - compute_haines_index() - Full upper-air formulation
   - compute_haines_index_surface() - Surface approximation
   - compute_haines_field() - Spatially-varying index
   
   Variants:
   - LOW: < 3000 ft elevation (950-850 mb)
   - MID: 3000-7000 ft elevation (850-700 mb)
   - HIGH: > 7000 ft elevation (700-500 mb)
   
   References:
   - Haines (1988) - Original index development
   - Werth & Ochoa (1993) - Operational validation
   - Mills & McCaw (2010) - Australian adaptation

3. BACKING FIRE ROS
   File: src/backing_fire_ros.H
   Lines: 240
   Purpose: Empirical backing fire rate of spread ratios
   
   Key Functions:
   - compute_backing_fire_ratio() - Wind-dependent ratio
   - compute_backing_ros_from_heading() - Direct ROS calculation
   - adjust_backing_coefficient() - Ellipse coefficient adjustment
   
   Empirical Ratios:
   - Light wind (< 5 mph): 30-40% of heading ROS
   - Moderate wind (5-15 mph): 15-25% of heading ROS
   - High wind (> 15 mph): 10-15% of heading ROS
   
   References:
   - Rothermel (1972) - Fire spread theory
   - Anderson (1983) - Ellipse model validation
   - Richards (1990) - Mathematical formulation

================================================================================
INTEGRATION GUIDE
================================================================================

To integrate features into main.cpp:

1. Include headers in wildfire_includes.H:
   #include "kbdi_index.H"
   #include "haines_index.H"
   #include "backing_fire_ros.H"

2. Add parameters to parse_inputs.H:
   struct KBDIParams { ... };
   struct HainesParams { ... };

3. Add MultiFabs to WildfireFields:
   MultiFab kbdi_mf;
   MultiFab haines_mf;

4. Call update functions in main simulation loop:
   if (inputs.kbdi.enable) {
       update_kbdi_field(f.kbdi_mf, temperature_mf, precip_mf, ...);
   }

5. Write to plotfiles (already in plot_results.H pattern):
   Add kbdi and haines to variable name list
