# Albini Spotting with 3-D Wind (massconsistent_amr) Regtest

Tests the Albini (1983) firebrand spotting model when driven by 3-D wind data
read from an AMReX plotfile produced by
[massconsistent_amr](https://github.com/hgopalan/massconsistent_amr).

## What is tested

- `albini_spotting.use_3d_wind = 1` activates 3-D wind reading.
- `albini_spotting.plt_wind_file = plt_wind_3d` points to the synthetic
  AMReX plt directory generated by `generate_plt_wind.py`.
- The reader (`plt_wind_reader.H`) opens the plt `Header`, parses the VisMF
  header, reads binary FAB data as flat 1-D arrays of `(x, y, z, u, v, w)`.
- Height-averaged `(u, v)` over all z-levels at each `(x, y)` column is
  bilinearly interpolated onto the fire-model 2-D grid and used for trajectory
  integration in place of the `vel` MultiFab.
- Spot fires appear downwind of the main fire front, deterministic with
  `random_seed = 42`.

## Domain

600 m × 600 m (UTM Zone 11N, 329900–330500 E, 3774900–3775500 N).
Grid: 64 × 64 cells (dx = 9.375 m).

The plt wind file (`plt_wind_3d/`) covers the same horizontal extent with
an 8 × 8 × 4 coarse grid (u = 5.0 m/s, v = 0.5 m/s, w = 0.0 m/s uniform).

## Generating the synthetic plt file

Run once before the simulation:

```bash
cd regtest/spotting/albini_spotting_3d_wind
python3 generate_plt_wind.py
```

## Running the test

```bash
cmake -S . -B build -DLEVELSET_DIM_2D=ON
cmake --build build -j
cd regtest/spotting/albini_spotting_3d_wind
python3 generate_plt_wind.py     # create plt_wind_3d/ if not present
../../../../build/levelset inputs.i
```

## Expected result

Spot fires appear east-northeast of the main fire front (consistent with
u = 5 m/s, v = 0.5 m/s wind). With `random_seed = 42` the result is
deterministic. The log should contain:

```
[plt_wind_reader] Loaded 256 3-D wind points from plt_wind_3d
[plt_wind_reader] 2-D column-averaged wind grid: 8×8
Albini spotting: using 3-D plt wind (8x8 2-D averaged) for trajectory
Albini spotting: generated N firebrands at step ...
```

## Physics verification (FM4, u=5 m/s, v_t=1 m/s)

- R ≈ 88 ft/min, I_B ≈ 9355 kW/m, H_z ≈ 257 m
- Flight time = 257 m / 1 m/s ≈ 257 s
- Mean landing distance ≈ 5 m/s × 257 s ≈ 1285 m (exceeds 600 m domain →
  most firebrands exit, but close-to-fire spot ignitions still occur)
